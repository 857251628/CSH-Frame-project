模块名称： Pop-up

使用方法

本功能是集成的开源fancybox1.3.4框架，使用此功能模块，需要在页面下面引入如下代码： 

<script type="text/javascript"  src="../images/jquery-1.8.3.min.js" charset="UTF-8"> </script>
<script type="text/javascript" src="../images/TRS.frame-v1.0.0.js" charset="UTF-8"></script>

引入说明

代码执行方法 
<script>
TRS.openWindow($('#js_box2'))
</script>

参数说明

例子代码

var obj = {
	'titlePosition'		: 'inside',
	'transitionIn'		: 'none',
	'transitionOut'		: 'none'
}
TRS.openWindow($('#js_box'),obj);  	//内嵌弹出窗体
TRS.openWindow($('#js_box2'));		//外连接弹出窗体
